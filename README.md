# 🛡️ Writeups de CTFs y Máquinas Vulnerables

Saludos! Este es un repositorio dedicado a la documentación de Writeups sobre máquinas vulnerables y laboratorios de plataformas como HackTheBox, TryHackMe y VulnHub.

El objetivo de este proyecto es practicar, reforzar y evidenciar conocimientos relacionados con:

* Enumeración de servicios
* Explotación de vulnerabilidades web
* Fuerza bruta y obtención de credenciales
* Explotación de CMS (Content Management System) vulnerables
* Escalada de privilegios en Linux
* Técnicas de Post-Explotación
* Uso de herramientas ofensivas comunes en Pentesting

Cada Writeup documenta paso a paso el proceso completo de explotación de la máquina, incluyendo reconocimiento, explotación y escalada de privilegios, acompañado de explicaciones e imágenes para facilitar el aprendizaje.
